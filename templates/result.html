推奨治療

{% if group.startswith('第1群') %}

推奨文：ERAまたはPDE-5阻害薬による単剤療法が推奨される（CQ1）。

解説：前毛細血管性PH（PAH）では肺血管拡張薬の単剤療法が第一選択。重症度や併存疾患に応じて選択する。

代表的臨床試験（SERAPHIN試験）要約：マシテンタンはPAH患者において疾患進行と入院リスクを有意に低下させた。PVRと6MWDにも良好な効果を示した。

{% elif group.startswith('第2群') %}

推奨文：左心不全の管理が最優先であり、PHに対する肺血管拡張薬の使用は推奨されない（CQ5）。

解説：左心負荷に起因するPHでは、肺血管拡張薬が悪化要因となる場合があるため禁忌とされている。

{% elif group.startswith('第3群') %}

推奨文：基礎疾患である肺疾患の管理が重要であり、肺血管拡張薬の使用は慎重に検討される（CQ6）。

解説：COPDやILD由来のPHでは、原疾患の増悪や換気血流ミスマッチの懸念があり、PH治療薬の効果は限定的。

臨床試験（INCREASE試験）要約：トレプロスチニル吸入薬はILD合併PH患者で6MWDと時間的予後指標の改善を示したが、広範な導入は慎重に判断すべき。

{% elif group.startswith('第4群') %}

推奨文：CTEPHに対してはバルーン肺動脈形成術やリオシグアトが推奨される（CQ8）。

解説：血栓性PHでは外科的介入または内服薬（sGC刺激薬）による介入が長期予後を改善する。

臨床試験（CHEST-1試験）要約：リオシグアトはCTEPH患者における6MWDとPVRの有意な改善を示した。再発防止にも有用とされる。

{% else %}

治療推奨の対象外または明確な方針がない分類です。

{% endif %}
