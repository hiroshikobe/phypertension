{% if group.startswith('第1群') %}

推奨治療

推奨文：ERA（アンブリセンタンなど）やPDE-5阻害薬（リオシグアトなど）による肺血管拡張療法が推奨される（CQ1: 推奨度A, CQ2: 推奨度A）。

  • 在宅酸素療法（必要時）
  • ERA（アンブリセンタン）
  • PDE5阻害薬（リオシグアト）
  • 呼吸リハビリテーション

解説：PAH（第1群）では前毛細血管性PHが対象であり、肺血管拡張薬（ERAやPDE-5阻害薬）が一次治療として推奨される。低酸素血症を伴う場合は酸素療法、活動耐容能低下に対してはリハビリも考慮される。

臨床試験概要：
SERAPHIN試験（マシテンタン）：PAH患者742例を対象とした無作為化比較試験で、マシテンタン10mg群は主要複合エンドポイント（死亡または疾患進行）のリスクを45%低下させた。長期予後改善を示し、PVR・6MWD・NT-proBNPの有意改善も報告された。

AMBITION試験：ERAとPDE-5阻害薬の初期併用療法が単剤療法より有効であることを示した国際多施設試験。

{% elif group.startswith('第3群') %}

推奨治療

推奨文：基礎疾患の管理を最優先としつつ、以下の治療が検討される（CQ6: 推奨度B, CQ7: 推奨度C）。

  • 在宅酸素療法（SpO₂低下時）
  • 吸入トレプロスチニル（ILD合併例に限り）
  • 呼吸リハビリテーション

解説：第3群（肺疾患関連PH）では原疾患の治療が最優先であり、肺血管拡張薬の使用は換気血流ミスマッチのリスクから慎重な適応判断が求められる。

臨床試験概要：
INCREASE試験（トレプロスチニル）：ILD合併PH患者326例において、吸入トレプロスチニル群はプラセボ群に比べ、16週時点で6MWDを有意に改善（+31m, p<0.001）し、NT-proBNP低下・機能分類改善・時間的複合エンドポイントでも有意差を示した。

{% elif group == '第1群と第3群の鑑別が困難' %}

推奨治療

推奨文：PAHと肺疾患関連PHの両側面を考慮し、以下の治療を症例ごとに検討（CQ1, CQ2, CQ6）

  • 在宅酸素療法（SpO₂低下時）
  • 吸入トレプロスチニル（ILD合併例）
  • ERA（アンブリセンタン）やPDE5阻害薬（リオシグアト）
  • 呼吸リハビリテーション

解説：第1群・第3群の鑑別が困難な症例では、肺血管拡張薬の効果とリスクを慎重に評価しつつ、酸素療法や支持療法を組み合わせた個別化医療が推奨される。

代表試験：
AMBITION試験：併用療法の有効性を示す。
INCREASE試験：ILD合併例でのトレプロスチニル有効性。
PATENT試験（リオシグアト）：6MWDおよびPVR改善を証明。

{% elif group.startswith('第2群') %}

推奨治療

推奨文：左心不全の管理が最優先であり、肺血管拡張薬の使用は原則推奨されない（CQ5: 推奨度A）。

解説：左心系疾患由来のPHでは、肺血管拡張薬が肺うっ血や左室前負荷を増悪させるため禁忌とされることが多い。心不全治療の最適化が本質的管理戦略。

{% elif group.startswith('第4群') %}

推奨治療

推奨文：慢性血栓塞栓性肺高血圧症（CTEPH）に対しては、手術または薬物療法（リオシグアト）が推奨される（CQ8: 推奨度A）。

  • 肺動脈血栓内膜剥離術（PEA）
  • バルーン肺動脈形成術（BPA）
  • リオシグアト（sGC刺激薬）

解説：CTEPHではPEAが根治的治療であるが、手術非適応例ではリオシグアトが唯一承認された選択肢。BPAも近年国内外で成績良好。

臨床試験概要：
CHEST-1試験（リオシグアト）：非手術適応CTEPH患者261例で6MWD+46mの改善、PVR-226dyn、NT-proBNP改善、WHO機能分類も優れた成績。安全性も良好。
